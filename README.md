## Como rodar este projeto?

```bash
# Clone este repositório
$ git clone Linkrepo

# Acesse a pasta do projeto no seu terminal
$ cd Pokedex

# Instale as dependencias
$ npm install

# Execute a aplicação
$ npm run start 

# A aplicação será iniciada na porta 3000, acesse pelo navegador: http://localhost:3000
```
